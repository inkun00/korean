# 애국가 전문가 교육용 자격증 발급 웹 앱 PRD

| 항목 | 내용 |
|---|---|
| 문서 버전 | v1.1 |
| 작성일 | 2026-06-22 |
| 제품명(가칭) | 애국가 탐험대 |
| 대상 | 초등학교 4~6학년 학생 및 담당 교사 |
| 수업 유형 | 창의적체험활동, 총 3차시 |
| 핵심 결과물 | 개인별 학습 점수, 등급, 명함 크기 교육용 디지털 인증서, 그룹별 애국가 뮤직비디오 |

## 1. 제품 개요

### 1.1 배경

초등학교 고학년 학생이 애국가를 단순히 암기하고 부르는 데 그치지 않고, 탄생 과정과 역사적 의미를 이해하며 가사의 뜻을 자신의 언어로 설명하고 정확히 부를 수 있도록 돕는 참여형 웹 수업 프로그램이다. 역사 탐험, 방탈출, 퀴즈, 음성 인식 노래 평가, 그룹 뮤직비디오 제작을 하나의 학습 여정으로 연결한다.

### 1.2 제품 목표

1. 학생이 애국가의 탄생 배경, 제작 과정, 인물 및 역사적 의미를 이해한다.
2. 학생이 애국가 가사의 핵심 어휘와 문장 의미를 설명하고 정확하게 부른다.
3. 학생이 모둠원과 협력하여 애국가의 의미를 시각적으로 표현한 뮤직비디오를 완성한다.
4. 모든 활동 결과를 자동 저장·점수화하고 성취 등급에 따른 디지털 인증서를 발급한다.
5. 교사가 별도 기술 지식 없이 학급, 차시, 과제, 결과물 및 학생 성취도를 관리한다.

### 1.3 성공 기준

- 참여 학생의 85% 이상이 3차시를 완료한다.
- 참여 학생의 80% 이상이 최종 60점 이상을 획득한다.
- 2차시 종료 시 가사 정확도 평균이 첫 시도 대비 15%p 이상 향상된다.
- 그룹의 90% 이상이 수업 시간 안에 뮤직비디오를 생성한다.
- 교사가 학급 개설부터 첫 수업 시작까지 10분 이내에 준비할 수 있다.
- 음성 인식용 녹음은 평가 처리 직후 폐기되어 별도 저장소에 남지 않는다.

## 2. 사용자와 역할

### 2.1 학생

- 교사가 제공한 학급 코드 또는 QR로 참여한다.
- 개인 학습 활동을 수행하고 자신의 점수, 피드백, 등급, 인증서를 확인한다.
- 3차시에는 모둠방에 참여하여 자신이 그린 애국가 관련 그림을 업로드하고 영상을 공동 제작한다.
- 다른 모둠의 비공개 그림 원본에는 접근할 수 없다.

### 2.2 교사

- 학교·학급·수업을 개설하고 학생 참여를 관리한다.
- 차시별 콘텐츠를 재생하거나 학생 기기에 배포한다.
- 학습 진행률, 퀴즈 및 방탈출 결과, 노래 평가, 그룹 결과물을 조회한다.
- AI 평가 결과를 재검토하고 필요한 경우 사유를 남겨 점수를 조정한다.
- 뮤직비디오 완성 여부를 확인하고 Padlet 게시용 결과물을 내려받는다.
- 학생의 학습 결과와 모둠 결과물을 다운로드하거나 삭제할 수 있다.

### 2.3 운영 관리자

- 영상, 슬라이드, 퀴즈, 방탈출 시나리오, 가사 해설 등 기본 콘텐츠를 관리한다.
- 부적절한 그림·영상 신고를 검토하고 게시를 제한한다.
- 학교별 이용 현황, 오류, AI 처리 실패율을 모니터링한다.
- 평가 기준, 등급 구간, 인증서 템플릿을 버전별로 관리한다.

## 3. 핵심 사용자 여정

1. 교사가 학급과 3차시 수업을 개설하고 학급 코드·QR을 발급한다.
2. 학생이 계정으로 로그인하거나 학교 정책에 따른 간편 계정을 생성해 학급에 참여한다.
3. 학생이 1차시 역사 학습과 방탈출을 완료하고 점수 및 해설을 확인한다.
4. 학생이 2차시 가사 의미 학습, 빈칸 채우기, 무작위 구간 부르기를 수행한다.
5. 시스템이 녹음 음성을 인식해 가사 정확도를 계산하고 재도전 피드백을 제공한다.
6. 3차시에서 학생들이 모둠방을 만들거나 배정받아 직접 그린 애국가 관련 그림을 업로드하고 영상 구성을 확인한다.
7. 시스템이 그림과 애국가 음원·가사 타이밍을 결합해 뮤직비디오를 생성한다.
8. 교사가 결과물을 확인하고 Padlet 게시용 파일 또는 공유 링크를 내려받는다.
9. 시스템이 누적 점수를 산정해 개척가·숙련가·전문가 등급과 수업 결과물용 인증서 이미지를 발급한다.

## 4. 수업 설계

기본 수업 시간은 차시당 40분으로 설계하되, 교사가 35~50분 범위에서 활동을 켜고 끌 수 있게 한다.

### 4.1 1차시: 애국가의 비밀을 찾아서 — 도입

#### 학습 목표

- 애국가가 만들어진 시대적 배경과 변화 과정을 말할 수 있다.
- 애국가의 작사·작곡 관련 인물과 역사 자료를 구분할 수 있다.
- 자료에 명확히 밝혀진 사실과 학설 또는 불확실한 내용을 구분한다.

> 콘텐츠 원칙: 작사자처럼 역사적으로 논쟁 또는 불확실성이 있는 내용은 단정하지 않고, 교육과정 및 공신력 있는 사료에 근거해 ‘확인된 사실’, ‘유력한 견해’, ‘알려지지 않음’을 명확히 표시한다. 콘텐츠 공개 전 역사·음악 교육 전문가의 감수를 필수로 한다.

#### 권장 진행

| 구간 | 시간 | 활동 | 시스템 기능 |
|---|---:|---|---|
| 흥미 열기 | 5분 | 오늘의 질문, 사전 퀴즈 | 즉시 응답, 익명 학급 통계 |
| 역사 학습 | 10분 | YouTube 영상과 슬라이드 시청 | 임베드 재생, 자막, 교사용 전체 화면, 대체 자료 |
| 비밀 탐구 | 5분 | 핵심 인물·사건·상징 카드 학습 | 카드 뒤집기, 용어 해설, 접근성 음성 안내 |
| 방탈출 | 15분 | 애국가 탄생 과정 가상 체험 | 스토리형 퍼즐, 단서, 타이머, 힌트 |
| 정리 | 5분 | 핵심 퀴즈와 오늘의 한 문장 | 자동 채점, 오답 해설, 결과 저장 |

#### 방탈출 구성: 시간보관소와 사라진 애국가 기록

- 방 1 ‘신문방’: 1896년 『독립신문』에 여러 애국가 가사가 있었음을 발견한다.
- 방 2 ‘황실악보실’: 1902년 대한제국 애국가와 현재 애국가를 구분한다.
- 방 3 ‘찬미가서가’: 1905년·1908년으로 다른 기록을 함께 인정한다.
- 방 4 ‘곡조복도’: 현행 가사가 한동안 「Auld Lang Syne」 곡조로 불렸음을 연결한다.
- 방 5 ‘새 선율 연구실’: 1935년 안익태가 새 곡을 작곡한 기록을 복원한다.
- 방 6 ‘임시정부 승인실’: 1941년 새 곡보 사용 허가 공표와 이후 음반·악보 기록을 찾는다.
- 방 7 ‘교과서 배포실’: 1948년 이후 학교와 공식행사를 통한 전국 보급을 이해한다.
- 방 8 ‘조사위원회실’: 1955년 조사에도 작사자가 확정되지 않았음을 인정한다.
- 퍼즐별 3단계 힌트를 제공하며 힌트 1회당 최종 점수에서 1점을 감점한다.
- 기록 신뢰도, 힌트 수, 불확실성 존중 선택에 따라 기록수호자·배운 탐험가·열린 역사 탐정 엔딩으로 나뉜다.
- 15분이 지나도 학습을 중단하지 않고 안내 모드로 계속 진행한다.

### 4.2 2차시: 애국가를 제대로 배우기 — 전개

#### 학습 목표

- 1~4절 가사의 주요 낱말과 구절의 의미를 설명할 수 있다.
- 애국가 가사를 정확한 순서로 완성할 수 있다.
- 무반주 상황에서 제시된 구간을 정확하게 부를 수 있다.

#### 권장 진행

| 구간 | 시간 | 활동 | 시스템 기능 |
|---|---:|---|---|
| 가사 탐구 | 10분 | 어려운 낱말·상징·구절 학습 | 터치형 주석, 쉬운 말 풀이, 이미지 카드 |
| 빈칸 채우기 | 10분 | 난이도별 가사 완성 | 드래그 앤드롭·직접 입력, 즉시 피드백 |
| 노래 도전 | 15분 | 무작위 절 또는 구간 무반주 녹음 | 구간 추첨, 카운트다운, 녹음, 음성 인식 |
| 성찰·재도전 | 5분 | 누락·대체 가사 확인 후 재녹음 | 단어 단위 피드백, 최고점 저장 |

#### 노래 평가 방식

- 학생이 ‘도전 시작’을 누르면 시스템이 학습 범위 안에서 절 또는 2~4개 행을 무작위로 선택한다.
- 3초 카운트다운 후 무반주로 녹음하며, 연습 1회와 평가 2회를 기본 제공한다.
- 음성 인식 결과를 정답 가사와 단어 단위로 비교해 정확도를 산정한다.
- 발음의 지역적·개인적 차이, 어린이 음성 특성, 배경 소음을 고려해 유사 발음 허용 사전을 적용한다.
- 음정이나 목소리의 아름다움은 기본 점수에 포함하지 않는다. 제품의 평가 목적은 가사 정확성이다.
- 점수가 낮거나 인식 신뢰도가 기준 미만이면 자동 낙점하지 않고 ‘다시 녹음’ 또는 ‘교사 확인’을 제공한다.
- 학생에게 원문, 인식문, 누락·대체 단어를 색상뿐 아니라 아이콘과 텍스트로 함께 보여준다.
- 녹음 데이터는 음성 인식과 채점 과정에서만 일시적으로 처리하며, 평가 결과가 생성되는 즉시 폐기한다. 별도의 음성 파일 저장소나 다시 듣기 기능은 두지 않고 전사 결과·정확도·점수만 저장한다.

### 4.3 3차시: 우리 반 애국가 뮤직비디오 — 표현·공유

#### 학습 목표

- 애국가 가사의 의미를 직접 그린 그림과 장면으로 표현한다.
- 모둠 내 역할을 나누고 디지털 결과물을 공동 제작한다.
- 다른 사람의 작품을 무단으로 사용하지 않고 온라인 게시 예절을 지킨다.

#### 권장 진행

| 구간 | 시간 | 활동 | 시스템 기능 |
|---|---:|---|---|
| 기획 | 7분 | 모둠 구성, 절·구절과 역할 배정 | 모둠방, 역할 카드, 스토리보드 |
| 자료 제작 | 13분 | 애국가 관련 그림 그리기·업로드 | 업로드, 자르기, 순서 변경, 기본 형식 검사 |
| 영상 생성 | 10분 | 그림과 애국가를 자동 편집 | 자동 타이밍, 전환, 자막, 엔딩 크레딧 |
| 검토·공유 | 10분 | 미리보기, 수정, 감상 | 재생성, MP4·썸네일 출력 |

#### 모둠방과 영상 생성 규칙

- 교사가 모둠을 자동 또는 수동 배정하며 학생 3~5명을 권장한다.
- 역할은 감독, 자료 담당, 순서 편집, 안전 확인, 발표로 제안하며 중복 배정할 수 있다.
- 업로드 대상은 학생들이 직접 그린 애국가 관련 그림으로 한정한다.
- 업로드 형식은 JPG, PNG, HEIC를 지원하고 파일당 최대 15MB, 모둠당 기본 20장으로 제한한다.
- 업로드 화면에 ‘애국가의 의미를 표현한 그림을 올려 주세요’라는 안내와 예시를 표시한다.
- 파일 형식과 용량을 검사하고 수업 주제와 명백히 관계없는 자료는 교사가 제외할 수 있다.
- 영상은 16:9, 1080p MP4를 기본으로 생성하며 저사양 환경을 위한 720p도 제공한다.
- 기본 템플릿은 가사 자막, 그림 전환, 시작 제목, 모둠명, 엔딩 크레딧을 포함한다.
- 영상 생성 실패 시 진행 상태를 보존하고 자동 재시도하며, 교사는 그림과 자막을 묶은 대체 슬라이드 영상을 즉시 출력할 수 있다.

#### Padlet 연동

- MVP에서는 완성된 MP4 파일과 썸네일을 내려받아 Padlet에 직접 업로드할 수 있게 한다.
- Padlet의 공식 연동 방식과 학교 계정 정책이 확인되면 후속 버전에서 ‘Padlet에 게시’ 기능을 제공한다.
- 게시 제목은 학생 개인 이름 대신 모둠 별칭을 기본값으로 사용한다.

## 5. 평가 및 교육용 인증

### 5.1 총점 구성

총점은 100점이며 각 영역의 원점수를 아래 비율로 환산한다.

| 평가 영역 | 배점 | 세부 기준 |
|---|---:|---|
| 역사·의미 학습 퀴즈 | 15점 | 사전·정리 퀴즈 중 정리 퀴즈 중심, 문항별 부분 점수 가능 |
| 방탈출 | 20점 | 퍼즐 정답 16점, 완료 4점. 힌트 1회당 1점 감점, 영역 최저 10점 보장 |
| 가사 의미 이해 | 15점 | 어휘·구절 의미 연결 및 설명 문항 |
| 가사 빈칸 채우기 | 15점 | 정답률 80%, 첫 시도 보너스 20% |
| 무반주 노래 가사 정확도 | 25점 | 최고 유효 시도의 단어 정확도 환산. AI 신뢰도 미달 시 교사 확정 |
| 뮤직비디오 협업·완성 | 10점 | 기획 2점, 역할 참여 2점, 가사·장면 연결 3점, 안전 준수 2점, 완성 1점 |
| 합계 | 100점 | 완료 상태와 함께 저장 |

#### 평가 공정성 원칙

- 속도는 방탈출 동점 해소 정보로만 활용하고 본점에 직접 반영하지 않는다.
- 장애, 발화 특성, 언어 환경 또는 기기 문제로 음성 평가가 불리한 학생에게 교사 확인, 낭독, 가사 순서 배열 등 대체 평가를 제공한다.
- 모둠 결과물 점수와 개인 참여 기록을 결합하여 무임승차 또는 기기 담당자 편중을 줄인다.
- 교사의 수동 조정은 조정 전 점수, 조정 후 점수, 사유, 일시를 감사 기록에 남긴다.
- 평가 기준이 바뀌어도 이미 발급한 인증서에는 당시 기준 버전을 유지한다.

### 5.2 등급 기준

| 등급 | 기준 | 의미 |
|---|---:|---|
| 개척가 | 0~59점 | 애국가 탐험을 시작하고 핵심 활동에 참여함 |
| 숙련가 | 60~84점 | 역사와 가사를 이해하고 대부분의 활동을 정확히 수행함 |
| 전문가 | 85~100점 | 역사·가사·노래·협업 활동에서 높은 성취를 보임 |

- 인증서 발급을 위해서는 점수와 별개로 3개 차시의 필수 활동을 모두 완료해야 한다.
- 미완료 학생은 현재 예상 등급을 볼 수 있지만 인증서는 발급되지 않는다.
- 교사는 특수한 수업 운영을 위해 등급 기준을 복제·수정할 수 있으나, 학급 시작 후에는 기준을 잠그고 학생에게 공지해야 한다.

### 5.3 수업 성취 인증서

본 인증서는 3차시 수업의 참여와 성취 결과를 기념하는 교육용 결과물이다.

- 명함 표준 비율 90×50mm, 인쇄용 300dpi PNG와 화면용 PNG를 제공한다.
- 포함 정보: 제품명, 인증 등급, 학생 표시명, 학급 또는 학교 표시명(선택), 발급일, 대표 캐릭터, 교사명 또는 기관명(선택).
- 전체 점수는 기본적으로 인증서에 표시하지 않고 학생 결과 화면에서만 제공한다.
- 등급별 만화풍 캐릭터와 테두리를 차별화하되 색상 외 배지 형태와 텍스트로도 구분한다.

## 7. AI 및 이미지 생성 요구사항

### 7.1 만화풍 이미지 생성

- 수업용 캐릭터, 역사 장면, 방탈출 배경, 가사 의미 카드, 등급별 인증서 장식 이미지는 요청된 `image2.0` 이미지 생성 모델을 사용한다.
- 실제 연동 시 모델 식별자는 공급자의 최신 API 명세와 사용 가능 여부를 확인하고 환경변수로 교체 가능하게 구성한다.
- 기본 스타일은 ‘밝고 친근한 교육용 2D 만화, 초등 고학년, 선명한 윤곽, 과도한 의인화나 전투 표현 없음’으로 통일한다.
- 역사 인물·태극기·무궁화·한반도 등 사실성이 필요한 요소는 전문가 검수와 상징물 표현 가이드를 통과해야 게시할 수 있다.
- 역사적 장면은 사진처럼 오인되지 않도록 ‘AI로 만든 학습용 삽화’라고 표시한다.
- 이미지 안에 가사나 중요 설명을 직접 생성하지 않고, 검수 가능한 웹 텍스트 레이어로 올린다.
- 프롬프트, 모델 버전, 생성일, 검수 상태, 사용 콘텐츠를 기록한다.
- 학생이 자유 프롬프트를 직접 입력하는 기능은 MVP에서 제공하지 않는다.

### 7.2 음성 AI

- 한국어 어린이 음성에 대한 사전 정확도 검증을 실시하고 학년·성별·지역 억양에 따른 오류 편차를 점검한다.
- 전사 정규화는 띄어쓰기, 문장부호, 허용 가능한 발음 변형만 처리하고 의미가 다른 단어를 임의로 정답화하지 않는다.
- AI 점수는 교사 판단을 보조하며, 최종 이의 제기와 수정 권한은 교사에게 둔다.
- 음성 입력은 전사와 채점 요청에만 사용하고 결과 반환 즉시 폐기한다. 서비스 서버, 파일 스토리지, 로그, 백업에 음성 파일을 저장하지 않는다.
- 외부 음성 인식 API를 사용할 경우에도 음성 데이터 미보관 및 모델 학습 미사용 옵션을 적용한다.

### 7.3 콘텐츠 안전

- 학생 업로드 대상은 직접 그린 애국가 관련 그림으로 제한한다.
- 주제와 관계없는 그림이나 수업에 부적절한 표현은 교사가 제외할 수 있다.
- AI 생성물과 학생 그림 모두 숨김·삭제 기능을 제공한다.

## 10. 비기능 요구사항

### 10.1 성능·신뢰성

- 일반 화면의 p95 로딩 시간은 학교 와이파이 기준 2.5초 이하를 목표로 한다.
- 퀴즈 채점은 1초 이내, 음성 전사·평가는 30초 이내에 결과를 제공한다.
- 3분 내외 1080p 영상은 요청 후 5분 이내 생성 완료를 목표로 한다.
- 한 학급 30명, 학교당 동시 10개 학급, 초기 전체 동시 사용자 3,000명을 기준으로 부하 시험한다.
- 자동 저장은 활동 중 10초 이내 또는 단계 완료 즉시 수행한다.

### 10.2 접근성·사용성

- 키보드만으로 주요 활동을 수행할 수 있고 화면 읽기 프로그램용 레이블을 제공한다.
- 색상만으로 정오답이나 등급을 구분하지 않는다.
- 본문은 초등학교 4학년이 이해할 수 있는 짧은 문장과 쉬운 용어를 우선한다.
- 영상 자막, 이미지 대체 텍스트, 음성 안내 대체 텍스트를 제공한다.
- 태블릿, Chromebook, Windows·macOS 데스크톱의 최신 주요 브라우저를 지원한다.
- 스마트폰은 인증서 확인과 간단 활동을 지원하되 3차시 편집은 태블릿 이상을 권장한다.

### 10.3 보안·데이터 처리

- 음성은 전사·채점 중에만 일시 처리하고 파일, 로그, 백업에 저장하지 않는다.
- 영상 재료는 학생들이 직접 그린 애국가 관련 그림으로 구성한다.
- 전송 구간과 저장된 학습 결과는 암호화하며 교사·운영자 권한에는 최소 권한 원칙을 적용한다.
- 점수 수정과 결과물 삭제는 감사 로그에 기록한다.
- 학생 간 순위표는 기본 제공하지 않으며 개인 점수는 본인과 승인된 교사만 조회한다.

### 10.4 저작권·콘텐츠 권리

- YouTube 영상은 임베드 및 교육 이용 조건을 확인하고 링크 중단에 대비한 대체 콘텐츠를 준비한다.
- 애국가 음원, 악보, 슬라이드, 폰트, AI 생성물의 이용·편집·배포 권리를 자산별로 기록한다.
- Padlet 등 외부 서비스 게시 전 해당 서비스 약관과 학교 계정 정책을 확인한다.
- 학생 그림은 해당 수업의 뮤직비디오 제작과 학급 Padlet 게시에 사용한다.

## 11. 분석 지표

- 차시별 시작률, 완료율, 평균 소요 시간, 중단 단계
- 퀴즈 문항별 정답률과 오답 분포
- 방탈출 퍼즐별 시도 수, 힌트 사용률, 완료율
- 음성 녹음 성공률, 재시도율, AI 신뢰도 미달률, 교사 수정률
- 첫 녹음 대비 최고 녹음의 가사 정확도 향상폭
- 그림 업로드 성공률, 영상 생성 성공률·평균 처리 시간
- 인증서 등급 분포 및 발급률
- 교사 수업 개설 시간, 검토 소요 시간, 재사용률

학생 개인을 비교하는 공개 순위 지표는 제공하지 않으며, 제품 개선 분석에는 가명 또는 집계 데이터를 우선 사용한다.

## 12. MVP 범위

### 포함

- 교사·학생 계정, 학급 코드 참여
- 3차시 콘텐츠 플레이어와 진행 저장
- 퀴즈, 가사 빈칸 채우기, 8개 기록방과 3개 엔딩으로 구성된 방탈출
- 브라우저 녹음, 한국어 음성 인식, 가사 정확도 평가, 교사 재검토
- 교사 지정 모둠방, 학생 그림 업로드, 스토리보드, 파일 형식 검사
- 한 가지 16:9 뮤직비디오 템플릿과 MP4 생성
- Padlet 수동 게시용 파일·문구 출력
- 100점 평가, 3등급 분류, 교육용 PNG 인증서
- 교사 대시보드, CSV 다운로드, 감사 로그
- 운영자 콘텐츠·이미지 자산 관리

### 제외 또는 후속 검토

- Padlet 직접 게시 API 연동
- 학생 자유 프롬프트 기반 이미지 생성
- 음정·박자·가창력 평가
- 공개 학생 랭킹과 경쟁형 리더보드
- 학생 간 자유 채팅
- 네이티브 iOS·Android 앱
- 학부모용 독립 앱과 SNS 자동 공유

## 13. 출시 단계

### 1단계: 콘텐츠·평가 검증

- 초등 교사, 역사 교육 전문가, 음악 교육 전문가가 사실성·난이도·평가 기준을 검수한다.
- 4~6학년 각 1개 소그룹으로 문해 수준, 퍼즐 시간, 가사 인식 오류를 관찰한다.

### 2단계: 기술 프로토타입

- 2차시 음성 평가와 3차시 영상 생성의 기술 위험을 먼저 검증한다.
- 서로 다른 교실 소음, 기기, 마이크, 어린이 발화에서 정확도와 처리 시간을 측정한다.

### 3단계: 학교 파일럿

- 2~3개 학교, 학년별 최소 1학급에서 3차시 전체를 운영한다.
- 교사 개입 시간, AI 점수 수정률, 음성 비저장 여부, 수업 시간 내 완료율을 확인한다.

### 4단계: 정식 출시

- 장애 대응, 데이터 처리 계약, 콘텐츠 권리 문서, 관리자 운영 매뉴얼, 교사용 수업안을 완성한다.
- 파일럿 성공 기준을 충족한 기능만 P0 출시 범위로 확정한다.

## 14. 출시 승인 기준

1. 3차시 전체 핵심 경로를 학생 30명이 동시에 수행해도 데이터 손실이 없다.
2. 네트워크가 1분간 끊겼다가 복구되어도 퀴즈·방탈출·스토리보드 진행 상태가 복원된다.
3. 음성 평가의 단어 정확도와 신뢰도 기준이 어린이 표본에서 검증되고 교사 수정률이 10% 이하이다.
4. AI 평가 실패가 점수 0점으로 자동 확정되지 않으며 모든 실패에 재시도 또는 교사 검토 경로가 있다.
5. 음성 평가가 끝난 뒤 서비스 서버, 파일 저장소, 로그, 백업 어디에도 음성 파일이 남지 않는다.
6. 업로드 단계에서 학생 그림만 사용한다는 안내가 명확히 표시된다.
7. 학생은 다른 학생의 점수나 다른 모둠의 비공개 그림 원본을 URL 추측이나 화면 조작으로 조회할 수 없다.
8. 완성된 영상과 Padlet 게시용 결과물을 정상적으로 내려받을 수 있다.
9. 총점·등급·교육용 인증서가 동일한 평가 버전에서 일관되게 생성되고 재발급 결과가 동일하다.
10. 역사적 사실, 애국가 관련 콘텐츠 권리, 접근성에 대한 필수 검수가 완료된다.

## 16. 미결정 사항

아래 항목은 설계·개발 착수 전에 제품 책임자와 학교가 확정해야 한다.

1. 공식 서비스명과 교육용 인증서에 표시할 교사 또는 기관명은 무엇인가?
2. 학생 계정은 학교 SSO, 이메일 없는 간편 계정, 교사 일괄 생성 중 무엇을 우선하는가?
3. 1차시에서 다룰 역사 자료와 인물의 공식 출처 및 감수자는 누구인가?
4. 애국가 학습 범위는 1절 중심인가, 1~4절 전체인가?
5. 실제 사용할 `image2.0` 모델의 공급자, API 식별자, 비용 및 데이터 처리 조건은 무엇인가?
6. 사용할 애국가 음원·악보·폰트·영상 자료의 이용 권리는 확보되었는가?
7. 학생 그림과 완성 영상은 학기 종료 후에도 보관할 것인가?
8. Padlet은 학교별 계정으로 수동 업로드할지, 공식 연동을 추진할지 결정되었는가?
9. 서비스 예상 학교 수, 동시 접속자 수, 학생 1인당 목표 운영비는 얼마인가?
